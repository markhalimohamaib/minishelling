
->Readline library:

	readline — Reads one line from standard input with line editing and history support.

	rl_clear_history — Frees all history entries from memory.

	rl_on_new_line — Informs Readline that a new line has begun (for proper redisplay).

	rl_replace_line — Replaces the current input line with a new string.

	rl_redisplay — Refreshes the current input line display.

	add_history — Adds a command line to the history list.
------------------------------------------------------------------------------------------------------

->Standard I/O:

	printf — Formats and prints text to standard output.

	write — Writes bytes to a file descriptor.

	read — Reads bytes from a file descriptor.

	open — Opens or creates a file, returns a file descriptor.

	close — Closes an open file descriptor.

	access — Checks file accessibility (existence, permissions).
------------------------------------------------------------------------------------------------------

->Memory management:

	malloc — Allocates memory dynamically.

	free — Frees dynamically allocated memory.
------------------------------------------------------------------------------------------------------

->Process control:

	fork — Creates a new process (child).

	wait / waitpid — Waits for a child process to finish, retrieves its status.

	wait3 / wait4 — Same as waitpid but with resource usage reporting.

	execve — Replaces current process image with a new program.

	exit — Terminates the current process.

	kill — Sends a signal to a process.
------------------------------------------------------------------------------------------------------

->Signals:

	signal — Assigns a handler for a specific signal (simple version).

	sigaction — Advanced interface for signal handling.

	sigemptyset — Initializes a signal set to empty.

	sigaddset — Adds a signal to a signal set.
------------------------------------------------------------------------------------------------------

->Directory and file info:

	getcwd — Gets the current working directory.

	chdir — Changes the current working directory.

	stat / lstat / fstat — Retrieves file information (metadata).

	unlink — Deletes a file.

	opendir — Opens a directory stream.

	readdir — Reads entries from a directory stream.

	closedir — Closes a directory stream.
------------------------------------------------------------------------------------------------------

->Duplication and pipes:

	dup — Duplicates a file descriptor.

	dup2 — Duplicates a file descriptor into a specified target descriptor.

	pipe — Creates a unidirectional data channel (pipe) for interprocess communication.
------------------------------------------------------------------------------------------------------

->Error and TTY utilities:

	strerror — Returns a string describing an error code.

	perror — Prints a descriptive error message to stderr.

	isatty — Checks if a file descriptor refers to a terminal.

	ttyname — Returns the terminal name for a file descriptor.

	ttyslot — Returns the slot number of the terminal in /etc/ttys.

	ioctl — Performs low-level device control operations.
------------------------------------------------------------------------------------------------------

->Environment:

	getenv — Returns the value of an environment variable.
------------------------------------------------------------------------------------------------------

->Terminal control:

	tcsetattr — Sets terminal attributes (e.g. echo mode).

	tcgetattr — Gets current terminal attributes.
------------------------------------------------------------------------------------------------------

->Termcap library:

	tgetent — Loads terminal capabilities from termcap database.

	tgetflag — Gets a boolean capability.

	tgetnum — Gets a numeric capability.

	tgetstr — Gets a string capability.

	tgoto — Computes a cursor movement string.
	
	tputs — Outputs a termcap string with padding handling.
------------------------------------------------------------------------------------------------------
